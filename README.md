\# (C# 코딩) 로그인화면



\## 개요

\-C# 프로그래밍학습

\-1줄소개: 아이디와 비밀번호를 입력하여 로그인하는 로그인화면

\-사용한플랫폼: -C#, .NET Windows Forms, Visual Studio, GitHub

\-사용한컨트롤:label, textbox, button

\-구현한기능

placeholder로 입력창 안내하는 기능 구현

아이디와 패스워드입력하였을 시 성공, 실패 여부 확인
